* Javascript
* Node js
* npm packages
* MySql2
* Handlebarsjs

## Usage
````
 - npm install
 - add .env file
 - add DB_NAME = (database name here)
       DB_USERNAME = 'usually root'
       DB_PASSWORD = 'insert password here
       SECRET = 'insert secret phrase'
 - mysql -u root -p
 - run source db/schema.sql
 - exit mysql
 - run npm start
 ````
